LloydyWeb Through the Ages

Version 5 Revision History

v5.5 - April 2004

  • Reviews back to having it's own section again
  • And the shop site removed!
  • Instead, individually linked items to Amazon.co.uk in reviews and other places
  • Links to my albums at ofoto.co.uk for printing of photos
  • Improved code and CSS - including a little eye-candy for browsers that support PNG

v5.4 - January 2004

  • Lounge moved to new consolidated reviews section
  • Added online shop in association with Amazon.co.uk
  • Other minor cosmetic changes

v5.3 - September 2003

  • Standard width across site
  • Some code clean up - both in CSS and XHTML
  • New section indices

v5.2 - August 2003

  • Minor redesign - re-ordered front page in preparation for three separate news feeds - LloydyWeb, roobottom dot com, and oz tour 2003
  • Home section narrowed again for better viewing on 800x600 screens - after seeing some users still have this!
  • Further coding improvements

v5.1 - June 2003

  • Minor redesign - new logo
  • Helvetica replaces DIN as sites typeface
  • Home section reorganised and with wider design
  • Further coding improvements
  • Section navigation is now text based, reducing download times

v5.0 - February 2003

  • Total site revamp for 2003
  • Site reorganisation
    • Degree essays now within new section 'DOCS'
    • 'Profile' section merged together into expanded homepage
    • The remaining pages from the former 'Stuff' section split into GB and 3LIONS section
  • Extensive code clean-up - now XHTML and CSS compliant
  • Pinter friendly (for Browsers supporting print style sheets)
  • Backwards compatible with outdated browsers - with styling removed
