LloydyWeb Through the Ages

Version 6 Revision History

v6.6 - August 2005

  • Design is now scalable in browsers that support... okay all browsers except the usual suspect!
  • Upgraded to Movable Type v3.2
  • Comment forms no longer use tables for layout (horrah!)
  • RSS feeds now served via Feedburner.
  • Textile 2 text formattting now used for inputing reviews and blog entries. You too can use it when entering your own comments.
  • Photos - Thumbnails now show number of comments for each photo.
  • Homepage - Reordered, pulling my latest del.icio.us links, flickr photos, blog entries, review and the current entry on rdc*. Added charity links.
  • Improved section indicies (much better!)
  • There is no end to the code fiddling - should we just except that it happens and move on?

v6.5 - April 2005

  • Blog, Photos and Reviews sections now all powered by Movable Type, allowing (improved) commenting, dynamic updating of indices, and RSS feeds for these sections.
  • Refined design. Greater use of rounded boxes (to be seen in browsers that support CSS2.1 more fully) as well as new pull quote, captioning and lead-in paragraph styles.
  • Portfolio trimmed, renamed Growth and now with a greater focus on past achievements. Newer work will be showcased on FourTwo.net when it re-launches later in the year.
  • Links page re-ordered, improved design and additional links added.
  • Introduction of Access Keys for main site navigation
  • Is there no end to the code fiddling?

v6.3 - January 2005

  • Refreshed design - darker background being the most notable aspect, but also refined navigation and more colour used
  • Photos section no longer uses popups, photos now viewed within a slide show
  • Rewritten CSS, fixed some big IE problems
  • More XHTML code fiddling...

v6.2 - November 2004

  • Finished remaining un-styled pages on Blog, especially those relating to comment posting
  • Sections indices contain greater detail as to what's available in that section
  • Navigation now gives clearer indication of page location
  • High content pages include sponsored links powered by Google AdSense
  • Reviews section re-coded, far-lighter PHP code
  • Added Christmas a theme!
  • The usual XHTML code fiddling.

v6.1 - August 2004

  • Blog now powered by Movable Type
  • Navigational tabs

v6.0 - June 2004

Content

  • Blogger integration - you can now post your comments on an improved blog
  • Lots of pages have been updated or rewritten
  • 3lions - Euro 2000 and 2004 results added
  • Articles - New web design article added
  • Articles - Print version of my dissertation available for download
  • Photos - New specific section, many new photo albums added, as well as wallpapers to download

Design

  • Site designed to fit more comfortably next to FourTwo.net
  • Headings and main body copy: Georgia
  • All other text: Lucinda Grande (Mac OS X), or Verdana (everyone else)
  • Two column grid, three for reviews section

Site search

  • More prominent search box
  • Improved results: with relevance scores and percentages
  • Sitemap

Code

  • Total rewrite
  • Database driven reviews section
  • Pages validate as XHTML Strict - click on the links in the footer to confirm XTML and CSS validation
  • PHP server side includes
  • Greater support for browsers not supporting JavaScript
