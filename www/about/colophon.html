Code

  • This site is coded using valid XHTML 1.1 and CSS 2.1
  • Many pages contain Microformats to add an extra layer of semanticism:
    • hCard for contact details,
    • hCal for any event information,
    • hAtom for syndicated content,
    • xFolk for recent del.icio.us links listed on the homepage,
    • hReview for the reviews in my blog,
    • hResume for my CV,
    • as well as a number of elemental microformats: XFN, relTag, relBookmark,...
  • External RSS feeds pulled in with the help of SimplePie or MagpieRSS.

Blog

  • The blog is powered by Movable Type 3.3 from Six Apart. The following plugins are used extensively:
    • MTTextile 2 by Brad Choate,
    • MTSwitch by Arve Bersvendsen,
    • MTKeyValues by Brad Choate,
    • SmartyPants by John Gruber.
  • Supports Gravatars, created by Tom Werner.
  • Comments are CoComment enabled.
  • RSS feeds powered by Feedburner.

Third Party Tools

The following third-party services are used on LloydyWeb:

  • My photos are powered by the rather wonderful Zen Photo.
  • 'Of Interest...' links generated by del.icio.us.
  • Feed of my latest photos powered by Flickr.
  • Tracks I've recently listened to scrobbled by Last.fm.
  • List of upcoming events powered by upcoming.org.
  • Visitor statistics expertly broken down by Mint.
  • Site search powered by Web Side Story (formerly Atomz).

Production

  • Initial design and build of version 7.0.0 was completed over a number of months during the summer of 2006, in both Walsall, West Midlands and Palo Alto, California.
  • Made on a Mac. I use a MacBook Pro (Core 2 Duo) running MacOS X 10.4 Tiger.
  • Ideas realised in Fireworks 8, and hand coded with the support of Dreamwever 8. Both products are available from Adobe Systems, Inc.
